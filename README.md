# Corona-API-Client

# Installation:

##### Maven:


```xml
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
```
```xml
<dependency>
	    <groupId>com.github.jan-tennert</groupId>
	    <artifactId>Corona-API-Client</artifactId>
	    <version>1.0.0</version>
	</dependency>
```
